## Task Manager – Team Progress Tracking System

Task Manager is a collaborative project management tool designed to help development teams organize tasks, assign responsibilities, and track project progress in real time.

The platform allows administrators to create tasks and assign them to specific team members such as frontend developers, backend developers, or product designers. Each user receives a personalized dashboard displaying their assigned tasks in card format.

As tasks are completed, users can mark them using a checkbox, which automatically updates the task status. The system dynamically tracks progress and displays whether tasks are completed or still in progress.

Both administrators and team members can visualize productivity through progress charts, while filtering tools allow users to quickly view completed tasks, pending tasks, or tasks currently in progress.

This project demonstrates task assignment workflows, role-based dashboards, progress tracking, and interactive UI components commonly used in modern team collaboration platforms.

## Features

- Admin task creation and assignment
- Role-based dashboards for team members
- Task cards displaying assigned work
- Task completion tracking with checkboxes
- Automatic status updates (Completed / In Progress)
- Progress visualization using charts
- Task filtering (Completed, In Progress, Pending)
- Real-time team productivity monitoring

## Purpose of the Project

This project demonstrates how task management systems work in real development teams. It focuses on improving workflow visibility, accountability, and collaboration by providing clear task assignments and visual progress tracking.

The system models real-world team environments where project managers or administrators assign responsibilities while developers track and update their progress.
